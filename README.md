# UN-AD

광고성/가짜 리뷰를 걸러내고 실제 소비자 리뷰를 우선 보여주는 웹 앱입니다.

## 실행 방법

```bash
npm start
```

실행 후 브라우저에서 `http://localhost:3000` 접속.

## 사용 흐름

1. 리뷰를 한 줄에 하나씩 입력
2. `리뷰 분석` 클릭
3. `실제 소비자 리뷰만 보기` 옵션으로 필터링

## 참고

- 현재 버전은 텍스트 패턴 기반 필터입니다.
- 실제 운영 시에는 플랫폼 정책/법률을 준수하고, 모델 기반 분류(LLM)와 신고 시스템을 함께 쓰는 것을 권장합니다.
